Dit is een algemeen probleem: je hebt een mooie webpagina met mooie plaatjes en fantastische Flash animaties, en helaas, de <b>zoekmachines</b> nemen je <b>website</b> niet op hun <b>index</b>.<br />
<br />
Dit kan verschillende redenen hebben:<br />
<br />

<h2>De HTML code van je webpagina's heeft geen betekenis voor zoekmachines</h2>
Zoekmachines gebruiken simpele software programma's om je <b>webpagina's te indexeren</b>. Een webpagina die er voor het menselijk oog prima uitziet, kan totaal betekenisloos zijn voor zoekmachines.<br />
<br />
Zoekmachines kunnen niet de tekstinhoud zien die in de plaatjes (GIF, JPEG, PNG, etc.), Flash elementen, JavaScript of andere scripttalen of multimedia bestandsformaten staan. Wanneer je JavaScript links gebruikt voor website-navigatie, dan kan het zelfs zo zijn dat zoekmachines je webpagina niet eens kunnen vinden.<br />
<br />
<b>Oplossing</b>: Controleer je webpagina's met een <b><a href="https://www.webconfs.com/search-engine-spider-simulator.php" target="_blank">zoekmachine spider simulator</a></b>. Spider simulators laten je zien <b>hoe zoekmachines je website zien</b>.<br />
<br />
<h2>De HTML code van je webpagina bevat fouten</h2>
Sommige HTML errors kunnen voorkomen dat zoekmachine spiders je webpagina's indexeren. Hoewel sommige <b>zoekmachine spiders</b> kleine <b>HTML code fouten</b> wel aankunnen, zullen ook sommige HTML code fouten foutieve informatie naar zoekmachine spiders zenden. Bijvoorbeeld, je webpagina heeft een &lt;/body&gt;-tag of een &lt;/html&gt;-tag aan het begin van de pagina, dan kan de spider dit ook zien als het einde van de pagina en niet de werkelijke content/inhoud gespiderd hebben. .<br />
<br />
<b>Oplossing</b>: Controleer de HTML code van je webpagina's met een HTML validator tool (http://htmlhelp.com/tools/validator/).<br />
<br /> 
<h2>De HTML code van je webpagina's bevat niet de juiste elementen</h2>
Wanneer je een <a href="/artikelen/2007/12/hoe_krijg_ik_een_nieuwe_website_hoog_in_de_zoekmachine/">hoge ranking in de zoekmachine</a> wenst voor een bepaald keyword, dan moet dit keyword op de juiste plaatsen op je webpagina staan. Bijvoorbeeld, het helpt over het algemeen om het <b>keyword te gebruiken in de pagina titel</b> (&lt;title&gt;).<br />
<br />
Er zijn veel andere factoren/elementen die je webpagina moet bevatten om hoog te kunnen scoren in de zoekmachines.<br />
<br />
<b>Oplossing</b>: Controleer de <b>HTML code</b> met een tool als <a href="https://www.ibusinesspromoter.com/" rel="nofollow">IBP's Top 10 Optimizer</a>. IBP's Top 10 Optimizer vergelijkt je webpagina met hoog scorende webpagina's en vertelt je in detail <b>hoe je je website moet veranderen om een top 10 (hoge) ranking te krijgen</b>.<br />
<br />
<h2>Je webserver verzendt de verkeerde status codes</h2>
Sommige webserver verzenden de verkeerde status codes. Wanneer een zoekmachine spider een webpagina van je website aanvraagt, dan verzendt je server een response code. Dit zou de "200 OK" code moeten zijn. Helaas verzenden sommige servers een "302 moved", of zelfs een "404 not found" response code naar de zoekmachine spiders. Dit terwijl de webpagina normaal getoond wordt in de webbrowser. In dat geval zullen de zoekmachine spiders denken dat de pagina niet bestaat en zullen ze de webpagina niet indexeren.<br />
<br />
<b>Oplossing</b>: Gebruik een <a href="https://www.webconfs.com/search-engine-spider-simulator.php" target="_blank">zoekmachine spider simulator</a> om te bekijken welke response code je webserver verzendt naar de zoekmachines.<br />
<br />
<b>Conclusie: Als je een top ranking in Google (en andere zoekmachines) wilt, dan moet je zekerstellen dat zoekmachines je webpagina's niet verkeer begrijpen. Gebruik de tips hierboven om er zeker van te zijn dat de zoekmachines zien wat je ze wilt laten zien.</b><br />
